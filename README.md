# Customer Data Preprocessing Assignment

## Tasks Performed

### 1. Data Cleaning
- Filled missing values in age using median
- Filled missing values in city using mode
- Removed duplicate rows

### 2. Encoding
- Applied One-Hot Encoding on city
- Applied Label Encoding on gender

### 3. Feature Scaling
- Applied Min-Max Scaling
- Applied Standardization

## Explanation

Min-Max Scaling scales data between 0 and 1 and is useful for algorithms like KNN and neural networks.

Standardization transforms data to mean 0 and standard deviation 1, which is useful for models like SVM and linear regression.
